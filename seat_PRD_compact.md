# seat.html PRD — 교실 매직박스 (단일 HTML 파일)

## 개요
초등 6학년 교실용 자리변경 + 발표뽑기 통합 프로그램. 단일 `seat.html` 파일(HTML+CSS+JS 인라인). 외부 의존성 없음. localStorage로 데이터 영속화.

## 디자인 방향
- 테마: 놀이공원/마술쇼 — 다크 네이비 배경 + 네온 골드/시안 포인트
- 교실 TV(1920×1080)에 띄워도 "와!" 하는 비주얼. 1280×720 반응형 대응
- Google Fonts 사용 가능 (Noto Sans KR 등)

## 1단계: 기반

### 메인 화면
- 탭 네비게이션: [자리 변경] [발표 뽑기] [⚙️ 학생 관리]
- 학생 관리 화면에서 CRUD 처리

### 학생 데이터
```js
{ id, name, gender:"M"|"F", fixedSeat:null|번호, isPresent:true, pickCount:0 }
```
- 입력방식: ① 개별 추가(이름+성별) ② 일괄 붙여넣기("이름,성별" 줄바꿈 구분) ③ localStorage 자동 저장/불러오기
- 출석 토글 가능

### 교실 배치도
- 행×열 격자 설정 (기본 5열×6행, 변경 가능)
- 좌석 클릭으로 비활성화/활성화 토글
- 상단에 교탁 표시

## 2단계: 자리 변경

### 제약조건 설정 패널
**A. 성별 고정** — 3모드 선택:
1. 사용 안 함
2. 남녀 영역 분리 (좌/우)
3. 남녀 교차 짝꿍 배치

**B. 특정 학생 고정** — 배치도에서 좌석 클릭 → 학생 선택 → 📌 고정. 복수 가능.

### 셔플 알고리즘
- Fisher-Yates Shuffle + 제약조건(성별/고정) 충족 시까지 재시도(백트래킹)

### 셔플 애니메이션 (5단계 연출, 교실 TV용 긴장감)
1. **카운트다운** 3-2-1 (화면 전체)
2. **해체** — 이름표가 자리에서 떠오름
3. **셔플** — 중앙에서 빠르게 뒤섞임 (3~5초, 회전+속도변화)
4. **착지** — 하나씩 새 자리로 날아가 안착 (bounce)
5. **완료** — confetti/반짝이 효과
- 속도 조절 슬라이더 (3초~15초)

### 결과 화면
- 남학생(파랑계열) / 여학생(보라계열) / 고정(금색테두리) 색상 구분
- [다시 섞기] [인쇄] 버튼

## 3단계: 발표 뽑기

### 캡슐머신 UI
- 투명 원형 통 안에 색색 캡슐(학생 수만큼)이 담겨 있음
- 캡슐들이 통 안에서 실시간 움직임 (2D 물리 시뮬레이션, requestAnimationFrame + 간단 충돌)
- 캡슐에는 번호만 표시 (이름은 뽑힌 후 공개)

### 뽑기 방식 2종
1. **자동 뽑기** — 버튼 클릭 → 랜덤 선택
2. **캡슐 직접 선택** — 교사가 통 안의 캡슐 클릭

### 뽑기 애니메이션
1. 캡슐들 격렬히 섞임 (2~3초)
2. 선택된 캡슐이 출구로 떨어짐
3. 캡슐 열리며 이름 공개 (드럼롤 + 조명 효과 + 이름 확대)

### ⭐ 시크릿 모드 (핵심)
- **진입**: 화면 우측 하단 빈 영역 5연타 또는 `Ctrl+Shift+S`
- **기능**: 작은 오버레이 패널에서 다음 뽑기 당첨자를 미리 지정
- **작동**: 애니메이션은 평소와 100% 동일하게 재생. 캡슐 직접 선택 모드에서도 어떤 캡슐을 골라도 지정 학생이 나옴
- **옵션**: ☑1회 적용 후 자동 해제 / ☐수동 해제까지 유지
- **보안**: 화면상 시크릿 모드 표시 일절 없음. 콘솔에도 로그 없음

### 추가 기능
- 뽑기 히스토리 표시 (오늘 뽑힌 학생 목록)
- 중복 방지 토글 (이미 뽑힌 학생 자동 제외)
- 전체 리셋 버튼
